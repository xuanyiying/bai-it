# OpenEn

一个帮你读懂英文长句的 Chrome 扩展。

OpenEn 会自动把英文网页上的复杂句子拆成好懂的小块（chunking），标注语法结构，翻译生词——你只需要提供一个 LLM API key。

## 功能

- **句子分块**：自动识别网页上的英文长句，拆解成易读的语法块，标注翻译
- **生词本**：遇到不认识的词随手收录，自动记录出处和语境
- **学习记录**：追踪阅读历史，生成每周学习报告
- **壁纸生成**：把生词做成手机壁纸，随时复习

## 技术特点

- 纯本地运行，零后端、零登录
- 数据全部存在浏览器本地（IndexedDB）
- 直接调用 LLM API（支持 Gemini、OpenAI 兼容格式）
- 隐私友好：你的数据不离开你的浏览器
